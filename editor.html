How to Use This Editor:

  1. Fill in all the fields below to create your article
  2. Click "Generate JSON" to see the formatted code
  3. Copy the generated JSON from the output box
  4. Open articles.json in a text editor
  5. Paste your new article BEFORE the closing ] bracket
  6. Make sure to add a comma after the previous article
  7. Save the file and refresh the website
